# Olympic Font

## Installation

Pour utiliser la fonte Olympic dans ce projet, suivez ces étapes :

1. Téléchargez la fonte Olympic depuis [Best Free Fonts](https://bestfreefonts.com/olympic)
2. Renommez le fichier téléchargé en `olympic.ttf`
3. Placez le fichier `olympic.ttf` dans ce dossier (`public/fonts/`)
4. Redémarrez le serveur de développement si nécessaire

La fonte sera automatiquement chargée et appliquée à tout le site.

## Alternative

Si vous ne pouvez pas télécharger la fonte Olympic, le site utilisera automatiquement une fonte système de secours.
